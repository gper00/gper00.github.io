Membangun Portofolio yang Menarik Perhatian Rekruter

[Nama Anda]

Diterbitkan pada 15 Juni 2025 · 8 menit baca

Saat melamar pekerjaan sebagai software developer, ada tiga pilar utama yang dilihat oleh rekruter: CV, profil GitHub, dan tentu saja, portofolio. Dari ketiganya, portofolio seringkali menjadi penentu. Mengapa? Karena di sinilah Anda bisa menunjukkan, bukan hanya menceritakan, kemampuan Anda.

Namun, banyak developer terjebak dalam membuat portofolio yang hanya berupa daftar proyek. Mereka lupa bahwa portofolio yang hebat adalah sebuah **cerita**. Ia menceritakan perjalanan Anda, cara Anda berpikir, dan masalah apa yang bisa Anda selesaikan. Artikel ini akan membahas cara membangun portofolio yang memiliki narasi kuat.

1. Mulai dengan "Mengapa"

Setiap proyek dalam portofolio Anda harus memiliki alasan yang jelas. Jangan hanya menulis "Aplikasi To-Do List". Jelaskan *mengapa* Anda membuatnya. Mungkin Anda ingin belajar tentang *state management* di React, atau mungkin Anda frustrasi dengan aplikasi yang ada dan ingin membuat versi yang lebih baik.

"Orang tidak membeli apa yang Anda lakukan; mereka membeli mengapa Anda melakukannya." - Simon Sinek

Menjelaskan "mengapa" menunjukkan bahwa Anda adalah seorang pemecah masalah yang proaktif, bukan hanya seorang juru ketik kode.

2. Tunjukkan Proses, Bukan Hanya Hasil Akhir

Hasil akhir memang penting, tapi proses di baliknya jauh lebih berharga bagi rekruter. Ini menunjukkan bagaimana Anda menghadapi tantangan teknis dan non-teknis. Berikut adalah beberapa hal yang bisa Anda sertakan:

  • Tantangan Teknis: Apa bagian tersulit dari proyek ini? Bagaimana Anda mengatasinya? Apakah Anda harus belajar teknologi baru?
  • Pilihan Desain: Mengapa Anda memilih arsitektur A daripada B? Apa pertimbangan di balik pilihan database atau framework Anda?
  • Pelajaran yang Diambil: Apa yang akan Anda lakukan secara berbeda jika memulai proyek ini lagi hari ini?

Contoh Cuplikan Kode

Misalnya, Anda bisa menunjukkan bagaimana Anda menangani sebuah *hook* kustom di React.

javascript
import { useState, useEffect } from 'react';

function useWindowWidth() {
  const [width, setWidth] = useState(window.innerWidth);

  useEffect(() => {
    const handleResize = () => setWidth(window.innerWidth);

    window.addEventListener('resize', handleResize);

    // Cleanup function
    return () => window.removeEventListener('resize', handleResize);
  }, []); // Empty array ensures this effect runs only once

  return width;
}

Pada akhirnya, portofolio Anda adalah kesempatan terbaik untuk menunjukkan gairah dan keahlian Anda. Buatlah ia bercerita, dan pintu kesempatan akan lebih mudah terbuka.
