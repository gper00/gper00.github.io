Setup Saya

Banyak yang bertanya tentang perangkat dan perangkat lunak yang saya gunakan dalam pekerjaan sehari-hari. Ini adalah daftar lengkap dari semua yang saya gunakan dan sukai. Daftar ini akan terus saya perbarui seiring waktu.

Peralatan Koding

  • Editor Kode

    Visual Studio Code

    Editor utama saya selama bertahun-tahun. Ekosistem ekstensinya tak terkalahkan. Ringan, cepat, dan bisa dikustomisasi untuk apa saja.

  • Tema VS Code

    One Dark Pro

    Tema gelap yang nyaman di mata dengan kontras warna yang pas. Saya selalu kembali ke tema ini.

  • Font Koding

    JetBrains Mono

    Font yang dirancang khusus untuk developer. Sangat mudah dibaca dan memiliki *ligatures* yang fungsional.

  • Terminal

    iTerm2 (macOS)

    Alternatif terminal bawaan macOS yang jauh lebih kuat. Fitur split panes, profil, dan kustomisasi warnanya sangat membantu.

Perangkat Keras

  • Komputer Utama

    MacBook Pro 14" (M1 Pro, 16GB RAM)

    Mesin yang sangat bisa diandalkan. Performanya cepat untuk kompilasi kode dan daya tahan baterainya luar biasa.

  • Monitor

    Dell UltraSharp U2721DE (27")

    Monitor 2K (QHD) dengan reproduksi warna yang akurat dan koneksi USB-C, jadi saya hanya butuh satu kabel untuk display dan mengisi daya laptop.

  • Keyboard

    Keychron K2 (Mechanical)

    Pengalaman mengetik yang memuaskan. Ukuran 75% adalah kompromi yang pas antara fungsionalitas dan minimalisme di atas meja.

Aplikasi & Layanan

  • Browser

    Arc Browser

    Sebuah pendekatan baru pada browser web. Manajemen tab dan 'spaces'-nya mengubah cara saya bekerja.

  • Desain UI/UX

    Figma

    Standar industri untuk desain antarmuka. Saya gunakan untuk membuat mockup, prototipe, hingga aset grafis untuk website ini.

  • Catatan & Produktivitas

    Notion

    Otak kedua saya. Semua ide tulisan, manajemen proyek, hingga catatan personal, semuanya tersimpan di sini.
